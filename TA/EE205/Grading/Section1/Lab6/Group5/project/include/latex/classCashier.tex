Serves a customer by taking them off the queue, and possibly putting an order, if valid, onto the order stack. 


\begin{DoxyItemize}
\item Retrieves the customer from the queue, popping them off.
\item Receives the customer\textquotesingle{}s desired order items in string format\+:

$<$number-\/of-\/items$>$ $<$name-\/of-\/item-\/with-\/no-\/spaces$>$
\begin{DoxyItemize}
\item Multiple items are separated simply by a space
\end{DoxyItemize}
\item If the \mbox{\hyperlink{classCashier}{Cashier}} detects an order for an invalid item that is not on the menu, it must expel() the customer.
\item Must calculate the cost of all the items ordered, and charge the customer.
\begin{DoxyItemize}
\item It the customer does not have enough money to pay, do not charge the customer, but instead, expel() them.
\end{DoxyItemize}
\item Once paid for, the order items must be consolidated into an \mbox{\hyperlink{classOrder}{Order}}, which is tagged with the unique customer ID generated at the beginning of this function, and then push it onto the stack.
